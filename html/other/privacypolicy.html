Privacy Policy

What data of mine does DataDeer.net collect? What does it do with it?

Stuff we log:

We have a log of access stored as:
  • IP of requester
  • Time of request
  • HTTP response code
  • Received HTTP GET header
We have an error log with timestamps.
I guess you could count the hit counter as a log.

Some stuff we store:

For signing in, we store your username and a hash of your password.
For group chat messages, we store the username that sent it, which chatroom (if in a chatroom), and what text.
For board games, we store the person who made the game and the board data.
For the "remember me" cookie, we store a random key to know its you.
For the "datadeer finance" thing, we store the stuff you see, like the times or amounts.
For the "todo list" we store the stuff you put in.
We store shared files. If the URL to the file doesn't work, that should mean it was deleted.
We store the settings you put in.
We store your user icon and biography.
